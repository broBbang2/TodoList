# TodoList (TaskFlow)

심플한 **할 일 관리 대시보드**입니다.  
로컬 스토리지를 활용해 브라우저에 할 일 목록을 저장하고, 진행률 카드와 필터로 현재 상태를 한눈에 볼 수 있습니다.

## 주요 기능

- **할 일 추가 / 완료 / 삭제**
- **전체 / 진행중 / 완료 필터링**
- **전체 개수 / 완료 개수 / 진행률 카드 표시**
- 브라우저 **로컬 스토리지 기반 자동 저장**

## 화면 구성

- `/`  
  - TaskFlow를 소개하는 **랜딩 페이지**
  - 좌측: 설명/버튼, 우측: 대시보드 **예시 카드 UI**
  - `대시보드 시작하기` 버튼으로 `/dashboard` 이동

- `/dashboard`  
  - 헤더에서 오늘 날짜와 간단한 안내 문구 표시  
  - 할 일 입력 폼, 필터 버튼, 할 일 목록, 통계 카드로 구성된 메인 대시보드

## 기술 스택

- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 에 접속하면 앱을 확인할 수 있습니다.
